Sistema Interno VMT 2.0

El VMT 2.0 de Kuehne+Nagel es un sistema interno que tiene como propósito evaluar el desempeño de la empresa con la finalidad de tener una mejora continua en los procesos que se realizan para lograr brindar un servicio de alta calidad.

El VMT 2.0 permite realizar la evaluación de una forma eficiente y eficaz, además de que es una gran herramienta de apoyo para los líderes que se encuentran dentro de la organización, con este sistema se pueden realizar planeaciones estratégicas más precisas para el cumplimiento de objetivos que permiten alcanzar las metas planteadas.

Aportaciones

Al realizar el VMT 2.0, nos encontramos con varios retos que hacían bastante complicada la interfaz de usuario.


Por eso ADTV aportó una serie de mejoras para lograr una experiencia de usuario mucho mas intuitiva y sencilla

Cuadro de Autoevaluación

 

Al acceder al sitio del VMT 2.0 en el dashboard principal se implementó una ruleta de comportamientos para ser evaluados por el usuario y enviarlos a su superior para que este a su vez lo evalúe también, la mejora visual implica además en dividirlos por colores que se van iluminando conforme se va terminando la evaluación por ambas partes.

Diagrama de Visualización de Personal

 

Para poder observar el progreso del personal del usuario, ADTV incorporó un sistema de visualización de equipo, de forma que conforme se vaya seleccionando a una persona, esta aparece en la parte superior y se despliega el siguiente nivel de subordinados.
También el color del recuadro en el que se encuentren indica si ya fueron enviados sus reportes y un pequeño ícono que muestra que tipo de performer es.

Barra Superior de Navegación

 

En el VMT 2.0 se hacen numerosas consultas por fecha y a diferentes miembros del personal de Kuehne+Nagel en cada una de sus secciones, por lo que ADTV implementó una barra superior de navegación que permite al ususario seguir estas consultas de manera mucho mas sencilla y poder observar los aspectos que se están avistando, de esta forma, no se necesita volver a una ventana espeífica para cambiar los filtros de busqueda.

Videotutoriales para cada sección

 

Para la adpatación al nuevo sistema, implementamos un video tutorial para cada sección, de manera que los empleados puedan aclarar sus dudas con respecto al VMT 2.0 y puedan acceder a los videos cuantas veces sea necesario

Notificaciones por medio de tarjetas

 

Debido a que el dashboard principal debía mostrar aspectos específicos y el usuario aún necesitaba consultar otros varios factores importantes, se incoporó una serie de notificaciones por medio de tarjetas en cada sección.
De esta forma el usuario podía ver de manera mas fácil y rápida los cambios importantes sin necesidad de navegar por el sitio completo.
Al seleccionar cada tarjeta se desplegaba en forma de pop-up el recueadro, la tabla, o lo que correspondiera a dicha consulta.

Gráficas dinámicas para la consulta de bonos y desempeño

 

En my Balance Score Card se hacen revisiones de desmpeño, divididos en “Quarters” (Q), lo que quiere decir que el periodo de corte es cada cuarto de año, o sea cada 3 meses, en adición hay 3 aspectos principales que conforman estas gráficas.
Para evitar generar varías páginas de gráficos, ADTV implementó un panel de filtros para que de esta manera el usuario pueda realizar varias consultas y hacer las comparaciones pertinentes en la misma ventana
